# Lead Magnets — Quick Start (Next 24 Hours)

## Today: Create Hidden Gems Airtable

1. **Go to airtable.com**
   - Sign up (free)
   - Create new base: "Tampa Hidden Gems"

2. **Set up fields** (copy-paste this structure):
   ```
   Name (Text)
   Neighborhood (Single Select)
   Category (Single Select: Bar, Restaurant, Cafe, Shop, Activity)
   Vibe (Text)
   Address (Text)
   Price Range (Single Select: $, $$, $$$, $$$$)
   Notes (Long Text)
   Why It's Hidden (Text)
   ```

3. **Import starter data**
   - Download `HIDDEN_GEMS_STARTER_DATA.csv` (above)
   - In Airtable: Extensions → CSV Importer
   - Upload the CSV file
   - Maps fields automatically
   - Takes 30 seconds

4. **Share publicly**
   - Click Share button (top right)
   - "Create a shared link"
   - Copy the link
   - Example: `airtable.com/shr/xxxxxxxxxxxxx`

5. **Add to beehiiv referral program** (if using beehiiv)
   - Go to beehiiv dashboard
   - Settings → Growth Tools → Referrals
   - Reward section: paste the Airtable link
   - Copy: _"Get instant access to 50+ Tampa hidden gems — organized by neighborhood, vibe, and price. Locals only."_

## This Week: Create Event Calendar PDF

1. **Open Google Docs**
   - Create new document: "Tampa Event Calendar — Next 60 Days"

2. **Structure it like this:**
   ```
   TAMPA EVENT CALENDAR
   Next 60 Days (March 27 - May 25, 2026)

   MARCH 27 - APRIL 2

   Mar 27 (Thursday) — Katt Williams @ Benchmark Arena, 8 PM — Comedy — $$$
   Mar 28 (Friday) — Gasparilla Music Fest Starts @ Meridian Fields — Music — $$$
   Mar 29 (Saturday) — Cuban Sandwich Festival @ Centennial Park — Food — Free

   ...continue for 60 days...
   ```

3. **Get events from:**
   - Your last 5 newsletter issues (copy the digest + event sections)
   - Google Calendar search "Tampa events March 2026"
   - Your own knowledge of scheduled events

4. **Export as PDF**
   - File → Download → PDF Document

5. **Upload & Share**
   - Upload to Google Drive
   - Right-click → Share → "Anyone with link can view"
   - Copy the share link
   - Shorten with bit.ly
